### Step 1:

`git clone https://github.com/3drdsh3in/Course_Design_For_Accreditation.git`

### Step 2:

Create a .env file and set PASSWORD=Linux_Pwd

Run In WSL2: `code CLOPLO_Mapping.ipynb`
